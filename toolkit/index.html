Agent Toolkit

Comprehensive toolkit for building autonomous AI agents. MCP servers, evaluation frameworks, observability.

Installation

npx -y @anthropic-ai/create-mcp-app@latest my-mcp-server
cd my-mcp-server
npm install @iowarp/agent-toolkit

IDE Integration

Works with Claude Code, Cursor, and VS Code.

MCP Servers

15+ MCP servers with 150+ tools for HDF5, NetCDF, Slurm, Lustre, Darshan, and more.

agent-toolkit — Core MCP servers
Slurm MCP — Job scheduling
HDF5/NetCDF — Scientific data
Darshan — I/O profiling
Jarvis — CI/CD for agents

Evaluation & Observability

Evaluation framework for context quality. Jarvis for production deployment. OpenTelemetry tracing.

IOWarp

Context engineering for autonomous AI agents in scientific computing. NSF-funded (Award #2411318).
