MCP Servers

15 specialized MCP servers for scientific computing - 150+ tools

IOWarp's MCP server collection provides AI agents with direct access to HPC infrastructure, scientific data formats, and research workflows.

Quick Start​

# Install and run any MCP server
uvx agent-toolkit hdf5
uvx agent-toolkit pandas
uvx agent-toolkit slurm

Available Servers​

Data I/O (4 servers, 49+ tools)​

HDF5 MCP - 27 tools for HDF5 operations

Capabilities:

  • AI-Powered Features - LLM sampling for insights, progress reporting, user elicitation
  • High Performance - 100-1000x speedup with LRU caching, 4-8x faster batch processing
  • Streaming Support - Handle unlimited file sizes with chunked reading
  • Resource URIs - Access HDF5 files via hdf5:// scheme with wildcards
  • Parallel Processing - Multi-threaded batch operations and directory scanning

Installation: uvx agent-toolkit hdf5

GitHub: agent-toolkit-mcp-servers/hdf5

Pandas MCP - 15 tools for CSV data analysis

Capabilities:

  • Data Loading & Export - Multiple formats (CSV, Excel, JSON, Parquet, HDF5)
  • Statistical Analysis - Descriptive statistics, correlation analysis, hypothesis testing
  • Data Cleaning - Missing data handling, outlier detection, duplicate removal
  • Advanced Analytics - Groupby operations, pivot tables, time series analysis

Installation: uvx agent-toolkit pandas

GitHub: agent-toolkit-mcp-servers/pandas

ADIOS MCP - 5 tools for scientific data I/O

IDE Integration​

Cursor

Add to ~/.cursor/mcp.json:

{
  "mcpServers": {
    "hdf5-mcp": {
      "command": "uvx",
      "args": ["agent-toolkit", "hdf5"]
    },
    "pandas-mcp": {
      "command": "uvx",
      "args": ["agent-toolkit", "pandas"]
    },
    "slurm-mcp": {
      "command": "uvx",
      "args": ["agent-toolkit", "slurm"]
    }
  }
}

Documentation: Cursor MCP docs

Claude Code
# Add HDF5 MCP
claude mcp add hdf5-mcp -- uvx agent-toolkit hdf5

# Add Pandas MCP
claude mcp add pandas-mcp -- uvx agent-toolkit pandas

# Add Slurm MCP
claude mcp add slurm-mcp -- uvx agent-toolkit slurm

Documentation: Claude Code MCP docs

VS Code

Add to your VS Code MCP config:

"mcp": {
  "servers": {
    "hdf5-mcp": {
      "type": "stdio",
      "command": "uvx",
      "args": ["agent-toolkit", "hdf5"]
    },
    "pandas-mcp": {
      "type": "stdio",
      "command": "uvx",
