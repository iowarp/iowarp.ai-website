Hermes API SDK

Programming with Hermes Content Transfer Engine

Overview​

Hermes provides a clean API for buffering and staging data across multi-tiered storage hierarchies in IOWarp.

Repository: github.com/iowarp/hermes

Core API​

Initialization​

#include <hermes/hermes.h>

HERMES_INIT();  // Initialize Hermes

Blob API​

The fundamental Hermes API for data operations:

hermes::Bucket bkt("my_bucket");

// Put data
std::vector<int> data = {1, 2, 3, 4, 5};
bkt.Put("my_blob", data.data(), data.size() * sizeof(int));

// Get data
std::vector<int> result(5);
bkt.Get("my_blob", result.data(), result.size() * sizeof(int));

STL API​

Template-based API for C++ STL types:

#include <hermes/api/stl.h>

hermes::Bucket bkt("my_bucket");

// Put STL container directly
std::vector<int> data = {1, 2, 3, 4, 5};
bkt.Put<std::vector<int>>("my_blob", data);

// Get STL container
std::vector<int> result = bkt.Get<std::vector<int>>("my_blob");

CMake Integration​

find_package(Hermes CONFIG REQUIRED)
message(STATUS "found hermes at ${Hermes_DIR}")

# Link Hermes client library
target_link_libraries(your_app hermes::hermes_core_client)
