The Agentic Science Era

AI agents are evolving from chatbots to autonomous scientists. IOWarp provides the infrastructure they need.

The Challenge

  • Manual data wrangling across heterogeneous formats
  • Static storage hierarchies limiting agent access
  • No semantic understanding of scientific context
  • Fragmented tools requiring human orchestration

The Solution

Intelligence Layer

ClaudIO agent framework • Expert subagents • Natural language interface

Tooling Layer

Agent Toolkit • 15 MCP servers • 150+ tools for science

Storage Layer

Context orchestration • Multi-tier intelligence • Seamless data access

IOWarp enables AI agents to navigate, understand, and act on scientific data through context engineering. Funded by NSF Award #2411318 ($5M).

Agent Toolkit for Scientific Computing

15+ specialized MCP servers making scientific data and infrastructure accessible to AI agents

Quick Start

$ uvx iowarp-agent-toolkit mcp-server hdf5
$ uvx iowarp-agent-toolkit mcp-server slurm
$ uvx iowarp-agent-toolkit mcp-server arxiv
$ uvx iowarp-agent-toolkit mcp-server pandas
$ uvx iowarp-agent-toolkit mcp-server plot

Features

Works with any MCP client

Claude Code, Cursor, VS Code
