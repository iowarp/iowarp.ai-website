Four-Pillar Architecture

Complete context engineering for autonomous scientific computing

Ingest

Context Assimilation Engine (CAE)

Transform multi-modal scientific data into AI-ready context bundles. Automated format normalization, semantic enrichment, and quality validation.

Explore Ingest →
Move

Context Transfer Engine (CTE)

Intelligent hierarchical data movement with ML-driven prefetching. Seamless memory-storage tiering for massive datasets beyond DRAM.

Explore Move →
Query

Context Exploration Interface (CEI)

Natural language interface for autonomous scientific discovery. WarpGPT enables zero-coding complex analyses through conversational AI.

Explore Query →
Extend

Platform Plugins Interface (PPI)

Seamless integration with HPC schedulers (Slurm, PBS), cloud storage, scientific instruments, and sandboxed execution environments.

Explore Extend →

IOWarp MCPs: AI Tools for Science

15 Model Context Protocol servers, 150+ tools — Enable AI agents for scientific computing in seconds

# Instant AI for scientific computing
uvx iowarp-mcps hdf5    # Analyze HDF5 files
uvx iowarp-mcps pandas  # Process datasets
uvx iowarp-mcps slurm   # Submit HPC jobs
uvx iowarp-mcps arxiv   # Search papers
  • Works with: Claude Code, Cursor, VS Code, Gemini CLI
  • 15 specialized servers: HDF5, ADIOS, Pandas, Slurm, ArXiv, Darshan, and more
  • Zero configuration: Instant deployment with uvx
  • Production-ready: Deployed at LLNL, ANL, NERSC
  • Open source: Apache 2.0 license
