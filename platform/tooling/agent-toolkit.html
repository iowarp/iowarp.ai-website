Agent Toolkit

15 specialized MCP servers for scientific computing - 150+ tools

Quick Start

# One command to enable AI for science
uvx agent-toolkit hdf5
uvx agent-toolkit pandas
uvx agent-toolkit slurm

Data I/O MCPs

HDF5 MCP

27 Tools | Comprehensive HDF5 operations for AI agents

The flagship implementation showcasing the full power of FastMCP for scientific computing with AI-powered features.

Capabilities:

  • AI-Powered Features - LLM sampling for insights, progress reporting, user elicitation
  • High Performance - 100-1000x speedup with LRU caching, 4-8x faster batch processing
  • Streaming Support - Handle unlimited file sizes with chunked reading
  • Resource URIs - Access HDF5 files via hdf5:// scheme with wildcards
  • Parallel Processing - Multi-threaded batch operations and directory scanning
  • Discovery & Optimization - AI-assisted exploration and bottleneck detection

Installation: uvx agent-toolkit hdf5

IDE Integration

Cursor Setup

Add to your Cursor ~/.cursor/mcp.json:

{
  "mcpServers": {
    "hdf5-mcp": {
      "command": "uvx",
      "args": ["agent-toolkit", "hdf5"]
    },
    "pandas-mcp": {
      "command": "uvx",
      "args": ["agent-toolkit", "pandas"]
    },
    "slurm-mcp": {
      "command": "uvx",
      "args": ["agent-toolkit", "slurm"]
    }
  }
}

See Cursor MCP docs for more info.

Claude Code Setup

# Add HDF5 MCP
claude mcp add hdf5-mcp -- uvx agent-toolkit hdf5

# Add Pandas MCP
claude mcp add pandas-mcp -- uvx agent-toolkit pandas

# Add Slurm MCP
claude mcp add slurm-mcp -- uvx agent-toolkit slurm

See Claude Code MCP docs for more info.

VS Code Setup

Add to your VS Code MCP config:

"mcp": {
  "servers": {

      "type": "stdio",
      "command": "uvx",

    "pandas-mcp": {
