LAMMPS Molecular Dynamics

Atom trajectory analysis from LAMMPS simulation with Adios MCP

Demo Overview

Automated analysis of LAMMPS molecular dynamics trajectories stored in Adios BP5 format using natural language queries. This demo demonstrates how IOWarp enables researchers to analyze complex simulation data through conversational AI, eliminating the need for manual file parsing and custom script development.

Technical Architecture

Adios-MCP Service

Adios-MCP is a Model Context Protocol server that provides AI agents with the ability to read and query Adios BP5 files:

User Prompt ‚Üí AI Agent ‚Üí Adios-MCP ‚Üí Adios2 Library ‚Üí BP5 Data
                ‚Üì
         Code Generation ‚Üí Execution ‚Üí Visualization

Key Capabilities:

  • Direct integration with Adios2 C++/Python API
  • High-performance binary data reading
  • Metadata extraction and variable discovery
  • Multi-dimensional array handling
  • Time-step iteration support

Supported Operations

  • Data Reading: Variable extraction, time-step iteration
  • Metadata Queries: File structure, variable names, dimensions
  • Array Operations: Slicing, indexing, data type handling
  • Attribute Access: Simulation parameters, metadata tags

import adios2
import matplotlib.pyplot as plt
from mpl_toolkits.mplot3d import Axes3D

def plot_atom_trajectory(bp_file, atom_id, output_file):
    """Plot 3D trajectory of specified atom"""
    # Read BP5 file
    with adios2.open(bp_file, 'r') as f:
        positions = []
        for step in f:
            atom_pos = step.read('atoms/position')[atom_id]
            positions.append(atom_pos)
