ParaView Flow Visualization

3D pipe flow visualization via ParaView MCP

Generate complex 3D visualizations from Incompact3d pipe flow simulations using natural language commands. This demo showcases how IOWarp transforms scientific visualization from a multi-step manual process into a conversational workflow through ParaView-MCP.

User Prompt‚Äã

plus three orthogonal slice planes (X, Y, Z)."

The ParaView-MCP enabled agent automatically executes:

  1. üîç Data Loading

    • Parses pressure and velocity field arrays
    • Validates data structure and bounds

  2. üìä Isosurface Generation

    • Extracts pressure isosurface at threshold pp=0.1
    • Applies marching cubes algorithm
    • Generates 3D surface mesh

  3. ‚úÇÔ∏è Slice Plane Creation

    • Creates three orthogonal cutting planes
    • X-plane: Captures flow cross-section

    • Z-plane: Reveals longitudinal patterns

  4. üé® Visualization Composition

    • Combines isosurface and slice planes
    • Applies scientific color mapping
    • Renders final 3D visualization

Technical Architecture‚Äã

ParaView-MCP Service‚Äã

ParaView-MCP is a Model Context Protocol server that bridges AI agents with the ParaView visualization engine:

User Prompt ‚Üí AI Agent ‚Üí ParaView-MCP ‚Üí ParaView Engine ‚Üí Visualization

Key Capabilities:

  • Direct integration with ParaView's Python API
  • Automated filter pipeline construction
  • Intelligent parameter selection
  • Format-agnostic data loading
  • Real-time rendering coordination

Supported Operations‚Äã

  • Data Loading: BP5, HDF5, VTK, NetCDF formats
  • Filters: Isosurfaces, slices, streamlines, glyphs
  • Rendering: Surface, volume, contour visualization
  • Export: Images, animations, data extracts
